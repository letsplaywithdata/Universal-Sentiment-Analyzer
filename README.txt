Follow the following steps to successfully run the app on your browser:

1- In your terminal, navigate to the path of this folder.

2- Install the requirements.txt by running the following command: pip install -r requirements.txt

3. Once the dependencies have been installed, run the following command : streamlit run app.py

3- Once the file runs successfully, open your browser and paste the following URL: 

				http://127.0.0.1:8501/   (the port may vary, check your console)
					OR
				http://localhost:8501/	

4- You will now be able to access your app
